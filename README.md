# agent.No1
## 项目简介
基于 AstrBot 搭建的学习辅助 QQ Bot。
项目主要完成了模型接入、插件配置、知识库接入、服务器部署和日常维护工作。
## 功能
- RAG知识库问答
- 联网搜索
- 每日单词推送
- 插件扩展
- Q群管理（监控对话中是否出现屏蔽词，入群防人机验证...)
- 活跃气氛小游戏
## 技术栈
- Python
- AstrBot
- DeepSeek API
- Linux
- Ubuntu Server
- GitHub
- text-embedding-v3
- Qwen3-Reranker
